http

qué es

Es el protocolo de transferencia de hiper textos, normalmente usado para la comunicación entre cliente y servidor

cliente == nuestro navegador


servidor == donde se guarda la pagina
  • - dns: myserver.com
  • - ip: 123.123.123.123:8080
  • - fotos: /fotos
  • - datos: /datos
  • - archivos: /archivos
  • - etc:

- Nuestra página es un archivo estático de html que el servidor le manda a nuestro navegador
- Nuestro navegador, tiene una manera de hablar con servidores: fetch : ajax
- ¿cómo le mando info al server? -> fetch : ajax

qué es la s del https: "secure" : e2e encryption

como funciona:

  1. servidor abre un puerto para escuchar peticiones http
  2. cliente manda una peticion http al servidor
  3. servidor analiza peticion del cliente: verbos http
  4. servidor responde cliente

peticiones http

  • url: base_url + path_url
  • headers: metadata
  • body: la info que yo mando, si es GET entonces no se manda nada

verbos

  • GET: pedir cosas
  • POST: mandar cosas nuevas
  • PUT: sobreescribir cosas que existen
  • PATCH: editar cosas que existen
  • DELETE: para que una cosa ya no exista

hay muchos otros verbos o métodos, pero no se usan casi
